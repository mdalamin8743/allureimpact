# Allure Impact — Builder.io Setup

## First Time Setup
1. Go to https://builder.io and sign in
2. Your API key is already set: 1223b1f041f142f19142c068c0cf70ee
3. In Builder.io dashboard → Models → ensure "page" model exists
4. Go to Content → Pages → New Entry
5. Set URL to "/" for homepage
6. Drag and drop any registered component from the left panel
7. Edit all text, colors, content visually
8. Click Publish — changes go live instantly

## Available Components in the Editor
- Hero — full screen landing hero
- Stats Bar — numbers and labels row
- Services List — numbered services with hover effects
- Two Column Content — text + decorative box layout
- Text Block — simple text section
- CTA Banner — call to action full width
- Contact Form — form with contact info sidebar
- Page Hero — inner page top section
- Team Grid — team members cards
- Divider — gold line or spacer

## To Add New Pages
1. Builder.io dashboard → Content → New
2. Set the URL (e.g. /about, /services)
3. Build with drag and drop
4. Publish
